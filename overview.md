# Project Overview

## Project Name
Simple Calculator using Python

## Objective
The purpose of this project is to build a basic calculator that can perform simple arithmetic operations such as:
- Addition
- Subtraction
- Multiplication
- Division

The project demonstrates understanding of core programming concepts along with practical usage of Visual Studio Code, Git, and GitHub for version control and project hosting.

## Tools & Technologies Used
- Programming Language: Python
- IDE: Visual Studio Code
- Version Control: Git
- Repository Hosting: GitHub
