## 3️⃣ Version Control Workflow
| Step | Git Command | Purpose |
|------|-------------|---------|
| Initialize Repository | `git init` | Start version control |
| Track Changes | `git add .` | Stage all files |
| Commit Work | `git commit -m "Message"` | Save progress snapshots |
| Create Main Branch | `git branch -M main` | Standard project setup |
| Connect Remote Repo | `git remote add origin <https://github.com/Parth63-ai/CS-fundamental_assignment4>` | Link GitHub repository |
| Upload Code | `git push -u origin main` | Publish project online |

## 4️⃣ Multiple Meaningful Commits
